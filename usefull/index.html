Are you forced to use java on a CS course in uni?
Does ur assistant or assesor (or whatever that role is called) hate you?
Did the professor not tell shit about how to get everything to work?
Despite going coming so far by yourself (as in without help from uni) does jpackage still give stupid errors claiming it doesn't know what exe or app-image or deb means?
Well then fear not because this entry is just for you. This solution goes for not-so-intelligent IntelliJ, Eclipse, NetBeans or any other IDE (IDE doesn't matter, just added keywords :/ )

This is simply java being stupid and not giving proper error messages, to fix this, if you're on Windows (10/11, probably older works too) you need to install Wix3 and latest versions of wix-cli and tools, and if you're on linux you need to install rpm-build (or rpm-tools for arch linux) and dpkg.
Afterwards simply restart your IDE/Shell and run jpackage again.

Note: if it seems to be successful but never ending make sure you get rid of any extra args you fed it while trying to get it to work (like --scan or --add-modules), this happened to me and removing the extra args fixed everything.
